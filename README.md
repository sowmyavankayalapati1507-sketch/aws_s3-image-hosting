#  AWS S3 Image Hosting & Sharing App

##  Project Overview

This project demonstrates a secure image hosting solution using Amazon S3. It allows users to upload, store, and access images with proper access control.

##  Features

* Public image access via direct URL
* Private image protection
* Pre-signed URLs for temporary secure access
* Bucket versioning enabled

##  Technologies Used

* AWS S3
* AWS Management Console
* HTML, CSS

 ## Security

* Public access is restricted to the public folder
* Private images cannot be accessed directly
* Secure sharing using pre-signed URLs

## Results

* Public image is accessible via URL
* Private image shows Access Denied
* Pre-signed URL allows temporary access






##  Conclusion

Successfully built a secure AWS S3-based image hosting system with proper access control and sharing.
