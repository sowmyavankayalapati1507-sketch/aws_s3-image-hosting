# AWS S3 Image Hosting & Sharing App

##  Project Overview
This project demonstrates a secure image hosting solution using Amazon S3. It allows users to upload, store, and access images using public and private access controls.

---

##  Features
- Public image access via direct URL
- Private image security using bucket policies
- Pre-signed URL for temporary secure access
- Bucket versioning enabled

---

## Technologies Used
- AWS S3
- AWS Management Console
- HTML, CSS

---

##  Steps Performed
1. Created an S3 bucket in AWS
2. Uploaded images to the bucket
3. Configured bucket policies for public access
4. Enabled Block Public Access settings
5. Tested private access (Access Denied)
6. Generated pre-signed URL for temporary access
7. Created HTML page to display images

---

##  HTML File
The `index.html` file is used to display images from AWS S3 in a browser.
